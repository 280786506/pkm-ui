# 介绍

PKM-UI 是一款基于 Vue 3 和 TypeScript 的 UI 组件库。

这款组件库其实是我为了对VUE3有个基本的了解，也是为了对自己这些年来的技术经验的一个总结。

此组件库全程由本人亲手编写，由于是个人项目所以会将些许的个人兴趣爱好融入其中，因此不建议您将此 UI 库用于生产环境。
